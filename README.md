A modern and responsive mini-market web application built using **React.js** and **TailwindCSS 3.4.17**.
The project includes product search, category filtering, dynamic price filtering, and a fully functional add-to-cart UI.

---

## **Features**

### **Product Browsing**

* Browse fruits and vegetables in a polished grid layout
* Search products by name
* Filter by category (Vegetable / Fruit / All)
* Adjust maximum price using a dynamic range slider

### **Shopping Cart**

* Add-to-cart functionality using React `useState`
* Global cart count displayed in the header
* Cart updates instantly when clicking **Order Now**

### **UI / UX**

* Fully responsive layout for mobile and desktop
* Smooth-scroll navigation between sections
* Dark/Light mode toggle
* Modern TailwindCSS-based design

## **Getting Started**

### **1. Clone the Repository**

```bash
git clone https://github.com/mo-ahmedx/hami-minimarket-2.git
```

### **2. Navigate to Project Directory**

```bash
cd hami-minimarket-2
cd src
```

### **3. Install Dependencies**

```bash
npm install
```

### **4. Start the Development Server**

```bash
npm run dev
```

Visit the app in your browser at:

```
http://localhost:5173
```

---

## **Build for Production**

Generate a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
